What is a Function

A function is a relation between a set of inputs that each have one output. A function consists of an input, an operation, and an output. For example, in f(x) = x + 2, the f stands for function and can be replaced in different functions, the x is the input, and x + 2 is the operation that gives us the output. In addition, the notation f(x) means "y is a function of x".

Function Examples

1. Function f(x) = x + 1

This is a simple function where f stands for function. The x in this function is the input, and x + 1 is the operation that gives us the output.

2. Function f(x) = 2x - 3

This function represents a line where 2x - 3 is the operation to transform the input x into an output.

3. Function f(x) = -x^2 + 4x + 1

This quadratic function has a parabolic shape, with -x^2 + 4x + 1 defining how the input x is transformed into the output.

Definitions

Coefficient
A number that multiplies a variable. In the quadratic function f(x) = ax^2 + bx + c:
  • a is the coefficient of x^2.
  • b is the coefficient of x.
  • c is the constant term.
Vertex
The highest or lowest point on the graph of a quadratic function. This point is where the parabola turns and is also the maximum point if the parabola opens downwards or the minimum point if it opens upwards.
Axis of Symmetry
A vertical line that passes through the vertex of the parabola, dividing it into two mirror-image halves.
Roots (or Zeros)
The points where the graph of the quadratic function intersects the x-axis. These are the values of x that make f(x) = 0.
Parabola
The U-shaped curve that represents the graph of a quadratic function. It can open upwards or downwards.
Standard Form
The general form of a quadratic function written as f(x) = ax^2 + bx + c.
Factoring Form
A way to express a quadratic function as a product of its factors, written as f(x) = a(x - p)(x - q), where p and q are the roots.
Discriminant
Part of the quadratic formula inside the square root, given by b^2 - 4ac. It determines the nature of the roots of the quadratic equation:
  • Positive: Two real roots.
  • Zero: One real root (repeated root).
  • Negative: No real roots (two complex roots).
Roots (p and q)
The values of x where the quadratic function equals zero. In the factoring form f(x) = a(x - p)(x - q), p and q are these root values.
Maximum Point
The highest point on the graph of a quadratic function. It occurs when the parabola opens downwards (a < 0). At this point, the function has its greatest value.
